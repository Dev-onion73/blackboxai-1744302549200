# VIT Chennai Campus

## Project Overview
The VIT Chennai Campus project is a web application designed to provide students with quick access to various academic resources, announcements, and schedules. The application features a responsive user interface built with React and Tailwind CSS, allowing students to easily navigate through sections related to academics, examinations, and research. 

## Installation
To run this project locally, follow these steps:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/yourusername/vit-chennai-campus.git
   cd vit-chennai-campus
   ```

2. **Open `index.html` in a web browser:**
   Since this project does not have a build process or server setup, simply open the `index.html` file in your preferred web browser to view the application.

## Usage
After opening `index.html`, you will be presented with the homepage. You can navigate through various sections by clicking on the links provided in the sidebar:

- **Home**: Displays the latest academic information and notices.
- **Academics**: View the academic calendar and important dates.
- **COE**: Check examination schedules.
- **Research**: Access research resources and guidelines.

## Features
- **Responsive Design**: The application automatically adjusts to fit various screen sizes.
- **Interactive Sidebar**: Easily navigate through different sections using a collapsible sidebar.
- **Quick Links**: Access important resources quickly using the "Quick Links" dropdown in the header.
- **Dynamic Content**: Displays relevant information based on the currently selected section.

## Dependencies
This project utilizes external libraries for styling and functionality, which are included via CDN links in the `index.html` file:
- [React](https://reactjs.org)
- [React Router](https://reactrouter.com/)
- [Tailwind CSS](https://tailwindcss.com/)
- [Font Awesome](https://fontawesome.com/)
- [Google Fonts - Inter](https://fonts.google.com/)

## Project Structure
```

- **index.html**: The entry point of the application that loads the necessary scripts and styles.
- **App.js**: Contains the main React application logic, including routing and component definitions like `Header`, `Sidebar`, and content-specific components.
- **styles.css**: Provides custom styles to enhance the user interface, utilizing Tailwind CSS classes alongside custom rules.
